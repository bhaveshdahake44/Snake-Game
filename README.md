# 🐍 Snake Game

A classic Snake Game built using HTML, CSS, and JavaScript.

## 🚀 Features

- Smooth snake movement
- Food generation
- Score tracking
- Game Over detection
- Keyboard controls
- Responsive design

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (Vanilla)

## 🎮 How to Play

1. Open the game in your browser.
2. Use the Arrow Keys to control the snake.
3. Eat the food to increase your score.
4. Avoid hitting the walls or yourself.
